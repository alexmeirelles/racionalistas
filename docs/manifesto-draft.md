# Manifesto (Versão: Scanability + Jobs)

A maioria vive no piloto automático. Consome o raso. Escolhe por impulso. Justifica depois. Chama isso de intuição.

**Nós chamamos de ruído.**

Sabemos que a intuição é só o eco da mediocridade. Funciona no que é fácil. **Falha no que realmente importa.**

Racionalista não é quem pensa mais. É quem ousa estruturar o pensamento antes de agir.

Não é frieza. É disciplina.
Não é lentidão. É precisão.
Não é medo de errar. É respeito absoluto pelo custo do erro.

**Pensamento claro não é dom.**
**É infraestrutura.**
