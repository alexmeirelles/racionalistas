# Roadmap de Lançamento: Oktal + Racionalistas

**Data Alvo:** 1 de maio de 2026
**Filosofia:** Vibe coding cirúrgico. Racionalistas é o Topo do Funil (Mídia) e Oktal é a Ferramenta (Produto/Conversão).

---

## 🟢 1. O que já está PRONTO (Entregue)

### Racionalistas (A Mídia)
- **Base Técnica:** Repositório Next.js estabelecido utilizando a mesma arquitetura do Oktal.
- **Identidade Visual:** Design System "Dark/Teal" validado e aplicado via `globals.css` (Minimalista, focado em alta densidade de leitura, fontes serifadas).
- **Landing Page:** Manifesto manifesto publicado com os CTAs principais atrelados.
- **Ecossistema Wiki:** Catálogo dos 13 Modelos Mentais gerados com conteúdo hiperdetalhado (Ensaios densos de 3+ parágrafos), navegação fluida sem plugins de terceiros sujeitos as falhas e UI em perfeita sincronia com a Landing Page.
- **Navegação Global:** Arquitetura do site já acomoda a rota `/forum` no cabeçalho.

## 🟡 2. O que estamos CONSTRUINDO (WIP / Em Breve)

## 🔴 3. Planejamento Diferido (Pós-MVP / Para Futuro)

- **Novos Mentores no Oktal:** Congelado em 5 personas por agora. (Expansões de +15 mentores serão guardadas).
- **Instagram/Mídia:** As sete postagens iniciais serão geradas próximo ao lançamento para retroalimentar o ecossistema. Não focar na frente gráfica avulsa agora.
- **Refactoring Massivo de Chat no Oktal:** O core original do chat Socrático não será mexido para este Sprint de Lançamento e manterá sua funcionalidade comprovada atual.
- **Monetização Exclusiva do Racionalistas:** O site emana como mídia livre. Ele enviará clientes para o *Oktal Pro*, portanto assinaturas onerosas nele não entram no roadmap.
